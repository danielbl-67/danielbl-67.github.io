📝 Operaciones CRUD Básicas

INSERT: Añadir nuevos registros a una tabla especificando campos:

INSERT INTO usuarios (nombre, email)
VALUES ('Daniel', 'tuemail@example.com');

UPDATE: Modificar datos existentes (¡No olvides el WHERE!):

UPDATE usuarios
SET nombre = 'Daniel B.'
WHERE id = 1;

DELETE: Eliminar filas específicas de la base de datos:

DELETE FROM usuarios
WHERE activo = 0;

🔗 Consultas Multi-tabla (Joins Esenciales)

INNER JOIN: Cruza registros que existen obligatoriamente en ambas tablas:

SELECT u.nombre, p.total
FROM usuarios u
INNER JOIN pedidos p ON u.id = p.usuario_id;

LEFT JOIN: Muestra todos los de la izquierda, tengan o no relación con la derecha:

SELECT u.nombre, p.total
FROM usuarios u
LEFT JOIN pedidos p ON u.id = p.usuario_id;

📊 Agrupaciones y Totales

COUNT y GROUP BY: Cuenta cuántos elementos hay por cada categoría (ej: pedidos por usuario):

SELECT usuario_id, COUNT(*) as total_pedidos
FROM pedidos
GROUP BY usuario_id;

SUM y AVG: Calcula la suma total o el promedio de valores numéricos de una columna:

SELECT SUM(total) as ingresos_totales,
AVG(total) as promedio_gasto
FROM pedidos;

⚡ Rendimiento y Buenas Prácticas

Filtra columnas (No SELECT *): Pide solo los campos necesarios para ahorrar red:

SELECT nombre, email FROM usuarios WHERE activo = 1;

Índices de Búsqueda: Acelera los filtros en campos muy consultados (ej: emails):

CREATE INDEX idx_usuarios_email ON usuarios(email);
