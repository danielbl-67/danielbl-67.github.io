🚀 Filtros y Transformaciones (Java Streams)

Filtrar y mapear listas: Olvídate de los bucles for gigantes y anidados. Usa Streams para procesar colecciones en una sola línea fluida:

List<String> nombresActivos = usuarios.stream()
  .filter(u -> u.isActivo()) // Filtra la lista
  .map(Usuario::getNombre) // Transforma el objeto a String
  .toList(); // Recolecta el resultado final

⚖️ Comparación de Objetos (equals & hashCode)

El problema: En Java, el operador == compara referencias de memoria, no el contenido real de los objetos.

Solución Tradicional: Sobrescribir manualmente los métodos para que digan que dos objetos con el mismo ID son iguales.

Solución Moderna (Lombok): Deja que la librería genere todo el código repetitivo de forma limpia con anotaciones:

@Data // Genera Getters, Setters, toString y equals
@AllArgsConstructor
@NoArgsConstructor
public class Usuario {
  private Long id;
  private String email;
}

🔌 Inyección de Dependencias en Spring Boot

Buenas prácticas: Evita usar la anotación @Autowired directamente sobre los atributos (Field Injection), ya que dificulta las pruebas unitarias.

Recomendación oficial: Usa siempre inyección a través del constructor de la clase:

@Service
public class UsuarioService {

  private final UsuarioRepository repository;

  // Spring inyecta la dependencia aquí automáticamente
  public UsuarioService(UsuarioRepository repository) {
    this.repository = repository;
  }
}
